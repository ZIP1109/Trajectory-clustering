////////////////////////////////////////////////////////////////////Synthetic data////////////////////////////////////////////////////////////////////////////////////////

The synthetic dataset were stored in the folder'Synthetic data'
In the subfolders of 'synthetic data', 'D1', 'D2', 'D3' and 'D4' store the datasets contain a single cluster of different shapes and types, respectively. 
'D5' stores the dataset contains four clusters of different shapes and types.
The shapefile 'predefined region.shp' stores the predefined polygons

In each subfolder:
        The file 'flow.txt' stores the simulated bivariate flows.
        The file 'neighbors.txt' stores neighbors of each predefined polygon 
        The file 'Cluster_flow.txt' stores the origin and destination pairs of clusters


 //////////////////////////////////////////////////////////////////// Method////////////////////////////////////////////////////////////////////////////////
